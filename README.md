# Couseplate Frontend

## 기술 스택

- **Expo SDK**: 52.0.40
- **React Native**: 0.76.7
- **React**: 18.3.1
- **Expo Router**: 4.0.19

## 프로젝트 디렉토리 구조

```plaintext
couseplate-frontend/
│
├── app/                  # 주요 애플리케이션 코드
├── assets/               # 이미지, 아이콘 등 정적 자산
├── node_modules/         # 프로젝트 종속성 모듈
│
├── .gitignore            # Git 제외 항목 정의
├── App.js                # 프로젝트 진입점
├── app.json              # Expo 설정 파일
├── package.json          # 프로젝트 종속성 관리
└── package-lock.json     # 종속성 버전 관리
```


### 실행
npx expo start
